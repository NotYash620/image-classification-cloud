# Image Classification System (Cloud Deployed)

A deep learning based image classification system implemented using PyTorch and deployed on Google Cloud Platform.

## Overview
This project demonstrates an end-to-end pipeline for multi-class image classification using the CIFAR-10 dataset.

## Features
- CNN based deep learning architecture
- Model training, validation and evaluation
- Confusion matrix and classification report
- Cloud deployment on Google Cloud Platform

## Tech Stack
- Python
- PyTorch
- NumPy
- Matplotlib
- Google Cloud Platform
